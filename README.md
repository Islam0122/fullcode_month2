## Finance Bot

Telegram-бот для учета личных финансов с использованием **Telegram Bot API**, фреймворка **Aiogram**, базы данных SQLite и внешних API.

---

## 🎯 Цель проекта

Разработать Telegram-бота для учета доходов и расходов с возможностью:

* добавления транзакций
* просмотра статистики
* фильтрации данных
* администрирования
* автоматических уведомлений

Проект должен быть реализован поэтапно с применением FSM, БД, API, middleware и деплоя.

---

## 📌 Функциональные требования

### 1. Регистрация пользователя

* Автоматическая регистрация при первом запуске
* Сохранение user_id в БД

### 2. Добавление дохода

Пользователь:

* нажимает кнопку "Добавить доход"
* вводит сумму
* вводит описание
* данные сохраняются в БД

### 3. Добавление расхода

Аналогично доходу:

* сумма
* категория
* описание

### 4. Просмотр баланса

* Общий баланс
* Общая сумма доходов
* Общая сумма расходов

### 5. История операций

* Список последних 10 операций
* Форматированный вывод

### 6. Статистика

* Сумма за день
* Сумма за месяц

### 7. Админ-панель

* Доступ только для admin_id
* Просмотр количества пользователей
* Просмотр общей статистики

### 8. Интеграция API

* Получение курса валют через публичное API
* Отображение курса по команде

### 9. Планировщик

* Ежедневное напоминание внести расходы

### 10. Web Scraping

* Получение финансовых новостей с сайта
* Вывод 3 последних новостей

### 11. Middleware

* Логирование действий пользователя
* Проверка регистрации

### 12. Деплой

* Проект должен быть задеплоен на VPS или Heroku
* Бот должен работать 24/7
